# sbz

Losovač okruhů ke státnicím. Otevři `index.html` v prohlížeči (funguje i na mobilu)
a klepni na tlačítko – vylosuje se náhodný okruh s definicí, rozsahem a předměty.

- 29 okruhů ze dvou skupin (Matematika a TI, Programové systémy).
- Každý okruh přijde jednou za kolo, než se sada zamíchá znovu.
- Filtr nahoře omezí losování na jednu skupinu.
- Vše je v jediném souboru `index.html`, bez internetu a bez instalace.
